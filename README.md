# Installation
Clone the repository:  `git clone https://github.com/kortik0/truveler.git`

Navigate to the project directory: `cd truveler`

# Usage
Build the program: 
`
cargo build --release
`

Run the program: 
`
./target/release/truveler [path_to_scan_directory]
`
